# Roblox Clone - Полная система

Многопользовательская 3D платформа для создания игр в стиле Roblox.

## 🎮 Возможности

- **Мультиплеер в реальном времени** - играйте с друзьями в одной комнате
- **Создание и строительство** - размещайте объекты (кубы, сферы, платформы)
- **Кастомизация** - выбирайте цвета для объектов
- **Чат** - общайтесь с другими игроками
- **Редактор уровней** - создавайте свои игровые миры
- **3D графика** - использует Three.js для рендеринга

## 🚀 Быстрый старт

### Установка зависимостей

```bash
cd roblox-clone
npm install
```

### Запуск

```bash
# Запуск в режиме разработки (сервер + клиент)
npm run dev

# Или отдельно:
npm start        # Сервер
npm run client   # Клиент (Vite)
```

Откройте браузер: `http://localhost:3000`

## 📁 Структура проекта

```
roblox-clone/
├── client/              # Фронтенд (Three.js + Vite)
│   ├── index.html       # HTML страница
│   └── main.js          # Логика клиента
├── server/              # Бэкенд (Node.js + Socket.IO)
│   └── index.js         # Сервер и WebSocket
├── assets/              # Ресурсы (текстуры, модели)
├── scripts/             # Скрипты для игр
└── package.json         # Зависимости
```

## 🎯 Как играть

1. **Введите имя** в поле ввода
2. **Создайте комнату** или присоединитесь к существующей
3. **Перемещайтесь** используя клавиши WASD + Пробел/Shift
4. **Стройте** выбирая инструменты внизу экрана:
   - 🚶 Перемещение
   - 📦 Куб
   - ⚽ Сфера
   - 🟫 Платформа
   - 🗑️ Удалить объект
5. **Общайтесь** через чат слева внизу

## 🛠 Технологии

- **Frontend:**
  - Three.js - 3D рендеринг
  - Socket.IO Client - связь с сервером
  - Vite - сборка и дев-сервер

- **Backend:**
  - Node.js - серверная платформа
  - Express - HTTP сервер
  - Socket.IO - WebSocket для реального времени
